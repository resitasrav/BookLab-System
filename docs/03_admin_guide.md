# Yönetici / Akademisyen Kılavuzu

## 1. Kullanıcı Yönetimi
- Yeni kayıt olan kullanıcıları onaylayın veya reddedin.
- Toplu kullanıcı statüsü güncelleme imkanı.

## 2. Rezervasyon Kontrolü
- Tüm rezervasyonları merkezi panelden görüntüleyin.
- Çakışan rezervasyonları inceleyin ve gerekirse manuel müdahale edin.

## 3. Raporlama ve İstatistikler
- Laboratuvar ve cihaz kullanım istatistiklerini alın.
- PDF veya Excel formatında rapor oluşturabilirsiniz.

## 4. Bildirim Yönetimi
- Sistem e-postalarını ve uyarıları kontrol edin.
- Gerektiğinde toplu e-posta gönderimi yapabilirsiniz.
